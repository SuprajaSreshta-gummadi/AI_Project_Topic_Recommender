# AI_Project_Topic_Recommender
AI Project Topic Recommender using NLP, TF-IDF and Cosine Similarity

## Project Overview

The AI Project Topic Recommender is an NLP-based recommendation system that suggests suitable AI and Data Science project topics to students based on their interests, skills, and academic background.

The system uses Content-Based Filtering with TF-IDF Vectorization and Cosine Similarity to match student profiles with available project topics.

---

## Objectives

- Recommend personalized project topics to students
- Apply NLP techniques for text processing
- Convert text data into numerical vectors
- Calculate similarity between student profiles and projects

---

## Technologies Used

## How It Works

1. Student details are collected.
2. Project descriptions are processed using NLP.
3. Text data is converted into TF-IDF vectors.
4. Similarity scores are calculated.
5. Top matching projects are recommended.
